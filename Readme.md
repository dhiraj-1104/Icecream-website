## 📦 Installation

To set up and run this project locally, follow these steps:

1. **Clone the repository**
   ```sh
   git clone https://github.com/dhiraj-1104/Icecream-website.git
   cd Icecream-website
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run start
   ```
